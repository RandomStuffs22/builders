h1. Want to contribute?

* we prefer branches but will accept patches
* please make a separate branch/patch for each bug fix or feature
* there should be an "issue":http://github.com/railsbridge/builders/issues for any submission
* please don’t send pull requests just update the issue with the url for your fix (or attach the patch) when it’s ready
* contributions without tests won’t be accepted

If you have any questions send a message to hardbap.

We use "Shoulda":http//github.com/thoughtbot/shoulda , "Factory Girl":http://github.com/thoughtbot/factory_girl and "RR":http://github.com/btakita/rr/tree/master for testing.

Other gems/plugins we use:

* "authlogic":http://github.com/binarylogic/authlogic
* "is_taggable":http://github.com/giraffesoft/is_taggable
* "will_paginate":http://github.com/mislav/will_paginate

h1. Got a feature request or find a bug?

Please open an "Issue":http://github.com/railsbridge/builders/issues
